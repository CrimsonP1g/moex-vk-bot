# Чат-бот социальной сети «ВКонтакте» для мониторинга показателей фондового рынка (на примере Московской биржи)
Программный комплекс разработан в рамках курсовой работы студента, обучающего по направлению подготовки 38.03.05 «Бизнес-информатика» (Казанского федерального университета). 
Информационная система автоматизирует процесс сбора, долгосрочного хранения и визуализации ключевых индикаторов Московской биржи (IMOEX) с использованием бесплатного программного интерфейса IMOEX ISS.

# Основной функционал
  Интеграция с VK Long Poll API: обработка входящих пользовательских событий в режиме реального времени посредством интерактивного меню (экранных клавиатур).
  Многопоточный планировщик рассылок: автоматическая отправка ежедневных аналитических сводок подписанным пользователям по заданному графику (ежедневно в 08:00).
  СУБД на базе SQLite: регистрация учетных записей пользователей и ведение хронологического учета котировок с ограничением дублирования записей (составной ключ уникальности даты и тикера).
  Аналитический модуль: динамическая генерация графиков изменения индекса Мосбиржи (IMOEX) с использованием библиотеки `matplotlib` по запросу пользователя.

# Архитектура проекта
  main.py — диспетчер событий VK Long Poll API, логика интерфейса, потоковый планировщик (schedule, threading).
  moex_parser.py — модуль интеграции с ISS Московской биржи, извлечение котировок акций, валютных пар и индексов.
  db_manager.py — менеджер реляционной базы данных, обработка SQL-запросов к SQLite.
  config.example.py — конфигурационный шаблон структуры аутентификационных ключей.
